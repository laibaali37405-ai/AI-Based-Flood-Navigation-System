# AI-Based Flood Navigation System

An intelligent navigation solution designed to provide real-time updates and safe routing during flood situations using AI algorithms. This system helps users identify safe paths and avoid water-logged areas.

## 🚀 Project Overview
This project leverages AI to analyze flood data and provide navigation assistance. It is built to enhance public safety by offering smarter route planning during emergencies.
